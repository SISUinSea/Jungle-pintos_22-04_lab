# Collab Practice

이 폴더는 Git/GitHub 협업 연습용 임시 폴더입니다.

## 목적

- Issue 만들기
- Branch 만들기
- Commit 하기
- Pull Request 만들기
- Review 하기
- Merge 하기
- Conflict 해결하기

## 규칙

1. Pintos 실제 코드인 `pintos/` 폴더는 건드리지 않습니다.
2. 모든 실습 작업은 `collab-practice/` 폴더 안에서만 합니다.
3. main 브랜치에 직접 push하지 않습니다.
4. 작업은 issue 단위로 진행합니다.
